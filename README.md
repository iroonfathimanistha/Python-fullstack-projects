# Python Projects

This repository contains multiple Python projects created for learning, experimentation, and practice.

New projects will be added regularly.

## Projects Included

### 1. Gun Detection Project

A simple OpenCV-based object detection project using a Haar Cascade Classifier.

#### Technologies Used

* Python
* OpenCV (`cv2`)
* NumPy
* imutils

#### Features

* Webcam video capture
* Real-time object detection
* Haar Cascade classifier support
* Frame processing using OpenCV

---

This repository will continue to grow with more Python projects and experiments over time.

## Installation

Install required packages:

```bash
pip install opencv-python numpy imutils
```

## Run a Project

Example:

```bash
python project1.py
```
